# GitNodes trial run

## Overview

A throwaway project note so the graph has more than concepts and decisions in
it. Point GitNodes at this folder, open the graph, and you should see these few
notes connected: the ADR links to a concept, the concepts link to each other,
and shared tags cluster them.

Edit this note in the app, add a link to
[Knowledge graph](../concepts/knowledge-graph.md), and watch the edge appear.
